## Local development

To spin up a local environment for this project, follow the following steps:

-   make sure you have DDEV (and Docker) installed.
-   ddev start
-   An .env file will be generated, we don't know how and why but we think it is because of ddev???
-   ddev composer install
-   ddev composer update
-   ddev php artisan key:generate
-   ddev php artisan migrate --seed
-   ddev launch
